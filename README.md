# WEATHER CHECKER CLI

## SETUP

##### 1. Create a `.env` file  
##### 2. Add your Weather API Key :  
    WEATHER_API_KEY=your_api_key_here  
##### 3. Install requirements :  
        `pip install -r requirements.txt`  
##### 4. Run :  
        python3 main.py (country/city)  
        => example : `pythone3 main.py madrid`
